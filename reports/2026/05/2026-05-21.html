1 Claude Code

VS Code 1.121 — Claude Agent Auto Mode

VS Code 1.121 shipped yesterday with deep Claude Code integration. The new "Claude Agent Auto Mode" lets Claude execute code changes without permission prompts, monitored by a safety classifier. Combined with SSH remote agent sessions, you can now run Claude Code agents on dev servers while coding locally.

Enable via:

Settings → Extensions → Claude Code → Auto Mode: On

For iOS devs running Xcode builds on remote Mac minis, this finally enables fully autonomous agent workflows on build machines — trigger a full build-test cycle without switching context from your local editor.

2 Cursor

Cursor v0.45 Multi-Repo Context Engine

Cursor shipped its most-requested feature: a multi-repository context engine that tracks dependencies and references across up to 10 linked repos simultaneously. For iOS monorepo teams with separate SPM packages, shared libraries, and backend repos, the AI now understands your full dependency graph.

Configure via:

cursor repos link --path ../shared-lib --path ../api-server

No more "I can't see that file" errors when the agent needs cross-repo context. The engine indexes symbols, types, and API contracts across all linked repos so completions and refactors are globally consistent.

3 Codex

Codex v0.131 — TUI Overhaul + codex doctor

OpenAI pushed Codex v0.131 with a richer Terminal User Interface, native @mentions search across files/plugins, expanded remote workflows, and a new diagnostics tool codex doctor that audits your project configuration for agent optimization.

Run auto-fix to get the free efficiency boost:

codex doctor --fix

Auto-resolves common issues: missing context files, outdated SDK versions, and suboptimal memory settings. For teams already using Codex, this is a free 15–20% efficiency boost just from better configuration — no model upgrade needed.

Hot Industry Concepts

Patterns, paradigms & structural shifts worth understanding

3 Concepts

WebMCP — Every Website Becomes an Agent Tool

Google's WebMCP isn't just a product feature — it's a paradigm shift for how AI agents interact with the web. Instead of brittle scraping, websites expose structured "tool endpoints" that agents can call reliably. Think of it as REST APIs but for AI agents navigating UIs. The W3C proposal means this could become a universal standard within 18 months.

For developers, adding WebMCP metadata to your web apps is like adding SEO — except instead of search engines, you're optimizing for AI agents. Start with: expose your most common user workflows as structured tool definitions. Early adopters will gain outsized benefit as agent-driven traffic begins to surpass human-driven browsing.

Core AI — Apple's Framework Rebrand Signals iOS 27 Generative Push

Reports ahead of WWDC 2026 (June 8) indicate Apple is rebranding Core ML to "Core AI" in iOS 27. This isn't cosmetic — it reflects a fundamental expansion from traditional ML inference to generative AI, agentic workflows, and Apple's Foundation Models framework. The new framework will support third-party model integration (Gemini, Claude) at the system level alongside Apple's own on-device models.

For iOS developers, this means your ML code will likely need migration from Core ML APIs to Core AI APIs — start auditing your model pipeline dependencies now. Teams using Vision, NaturalLanguage, and CreateML frameworks should expect significant API surface changes at WWDC.

Agentic Finance — AI Agents as First-Class Financial Entities

Catena Labs' trust bank charter filing represents a new category: financial infrastructure purpose-built for AI agents. In 2026, agents need to pay for compute, purchase API access, manage subscriptions, and settle inter-agent transactions — but current banking rails assume human account holders.

Agentic finance infrastructure (Catena, Stripe's agent payment APIs, Coinbase's programmatic accounts) is emerging to serve this market. For developers building autonomous agent systems, the "how does my agent handle money" problem is being solved at the infrastructure layer — within 12 months, agent-native payment APIs will be as commoditized as SMS APIs are today.
