Daily Tech Tips — AI Infra & Prompt Engineering

Friday rotation: RAG pipelines, tool use patterns & agent architecture

3 Tips
1 MCP / Agent Infra

OpenAI Secure MCP Tunnel — Private Servers, Zero Public Exposure

OpenAI released Secure MCP Tunnel for enterprise customers (May 19). It allows Codex, AgentKit, and the Responses API to securely connect to private/on-prem MCP servers without exposing them to the public internet. For teams running internal tools behind firewalls, this eliminates the security risk of exposing MCP endpoints entirely.

Configure via:

codex mcp tunnel --target internal-server:8080 --auth enterprise-sso

This is the missing piece for regulated teams (fintech, healthcare, defense) wanting agentic workflows on private infrastructure. Pair with codex mcp list to verify tunnel registration before deploying agents that touch sensitive internal tools.

2 Gemini / Agent Hosting

Google Managed Agents API — Stateful Agents in Sandboxed Environments

Launched in public preview at I/O Day 2, Google's Managed Agents in the Gemini API lets you deploy stateful, autonomous agents running safely in isolated Linux sandboxes hosted by Google. Unlike self-hosted agents, these handle memory, state persistence, and execution isolation automatically — removing the "where does my agent run?" problem entirely.

Bootstrap a sandboxed agent:

gcloud ai agents create --model gemini-3.5-flash --sandbox=true

Ideal for RAG pipelines that need to call external tools, store intermediate results, and survive multi-turn sessions without developer-managed state backends. The sandbox includes read/write filesystem access, Python runtime, and outbound HTTPS — enough for most production agentic workflows.

3 Observability / LangChain

LangChain AgentOps — Production Monitoring for Autonomous AI Agents

LangChain released AgentOps toolkit for production-grade agent monitoring, debugging, and optimization. If you're running Claude Code, Codex, or custom agents in production workflows, AgentOps provides full observability: trace each agent decision, measure token costs per task, identify failure patterns, and optimize routing between models.

Install and instrument:

pip install langchain-agentops

Essential for teams scaling from "experimenting with agents" to "agents in production." The dashboard surfaces token burn per workflow step, tool-call failure rates, and latency outliers — the three biggest cost and reliability levers once you're beyond the prototype phase.

Intelligence Orchestration — The Developer's New Core Skill

In 2026, the highest-value engineering skill has shifted from writing code to orchestrating AI agents. Developers are becoming "intelligence orchestrators" — designing systems where multiple AI agents (Claude Code, Codex, Gemini) collaborate on different aspects of a codebase simultaneously. The job is no longer "write the code" but "design the agent workflow, define guardrails, route tasks to the right model, and verify outputs."

This requires understanding prompt engineering, tool-use patterns, RAG architecture, and agent governance simultaneously. The practical skill set: know which model is best for which task (Claude for reasoning, Codex for fast edits, Gemini for long context), how to write system prompts that constrain agent behavior reliably, and how to build evaluation pipelines to catch agent failures before they reach production.
