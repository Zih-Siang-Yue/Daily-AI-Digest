Monday Tips: Claude Code / Codex Agentic Workflow

Monday rotation: hooks, MCP tool registration, and AGENTS.md standing operating procedures

3 Tips
1 Claude Code

Wire Pre-Tool & Post-Tool Hooks to Guard Your Agentic Loop

Claude Code's lifecycle hooks let you intercept every tool call before and after execution — perfect for logging, enforcing guardrails, or auto-committing checkpoints. Drop a .claude/hooks.json in your repo with pre_tool and post_tool script paths. Your pre_tool.sh can reject dangerous shell commands (rm -rf), while post_tool.sh can run git add -p && git commit -m "[agent checkpoint]" after every file write. This gives you a full audit trail of what the agent touched — and lets you kill the loop safely if it veers off course.

{ "pre_tool": "scripts/pre_tool.sh",
  "post_tool": "scripts/post_tool.sh" }
2 Claude Code + MCP

Expose Your Internal APIs as MCP Tools — No Cloud Required

With Claude Code's MCP support, register any local or internal service as a first-class tool the agent can call autonomously. Add a server to .claude/mcp.json with the transport config. Now Claude Code can call mydb.query(sql) directly during an autonomous task — no copy-pasting SQL results, no context switching. The agent queries, inspects schema, and writes migration files in a single loop.

{ "servers": [{
  "name": "mydb",
  "command": "npx",
  "args": ["@myorg/mcp-postgres",
           "--conn", "$DATABASE_URL"]
}] }
3 AGENTS.MD

Use AGENTS.md as a Standing Operating Procedure for Your Codebase

Both Claude Code and Codex read AGENTS.md at the repo root as their persistent briefing document. Structure it with three sections: Architecture (describe your monorepo), Commands (test, lint, build), and Agent Rules (never commit to main, run tests after every task). Enterprises that standardize AGENTS.md report 60% fewer 'hallucinated architecture' errors in overnight agentic loops.

# AGENTS.md

## Architecture
Monorepo: /apps (Next.js), /packages (shared),
/services (Go microservices)

## Commands
- Test:  pnpm test
- Lint:  pnpm lint
- Build: pnpm build

## Agent Rules
- Never commit directly to main
- Run tests after every task
- Use conventional commits

Hot Concepts

Structural patterns & paradigm shifts surfacing this Monday

3 Concepts

Agent Budget Governance — The CFO's New AI Headache

As agentic coding tools mature, enterprises are discovering that token cost scales with agent autonomy — not developer headcount. An agent authorized to run overnight loops can exhaust monthly AI budgets in days. Microsoft's pullback from Claude Code and Uber burning its 2026 budget in four months are early signals of this structural problem.

The emerging discipline involves per-agent token quotas, embedding cost telemetry into CI/CD, and routing high-autonomy tasks through human approval gates when spend exceeds thresholds. Expect a new FinOps-for-AI tooling category: dashboards showing cost-per-PR, cost-per-feature, and ROI-per-agent-loop.

Agent Client Protocol (ACP) — The Missing Integration Standard

ACP governs agent-to-IDE communication: navigating file trees, opening diffs, executing terminal commands, surfacing results. Where MCP handles agent-to-external-tool, ACP handles agent-to-editor. Today each vendor implements this incompatibly — every IDE integration is bespoke and non-portable.

If adopted broadly, enterprises could swap agent models without re-integrating IDE toolchains. The governance model mirrors the MCP Consortium, with JetBrains seeding the reference implementation and publishing the spec openly for Cursor, Claude Code, Windsurf, and others to adopt. This is the missing layer between MCP and the developer experience.

Structural Spot Demand Collapse — The Hidden BTC Risk Signal

CryptoQuant's 'structural spot demand' metric tracks net real-money BTC buying independent of derivatives. At -147,000 BTC (2026 low) while price holds $77K, the divergence is historically unstable: it resolves via either a sharp drop when derivatives fade, or a sharp rally if a macro catalyst restores real buying.

For treasury managers, this metric matters for timing strategic entries. The current regime — price supported by derivatives while institutional spot demand is absent — means the Thursday PCE print carries outsized weight. A cooling read could restore the real-money buying pressure the market currently lacks.
