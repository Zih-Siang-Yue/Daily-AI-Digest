Cursor · Workflow Cursor's Background Agents: Queue Tasks While You Sleep, Review PRs in the Morning

Cursor's Background Agents — launched in v0.50 and now stable — let you queue multiple coding tasks that execute autonomously in cloud sandboxes while you're away. The workflow: open Cursor's agent panel, type a series of natural-language tasks ("Migrate all UIKit delegates to async/await", "Add unit tests for the PaymentService module", "Fix all SwiftLint warnings in /Sources"), and each spawns an isolated cloud environment that works independently. When you return, each task has produced a diff you can review and merge. The advanced pattern: chain tasks with dependencies using @after:task-1 annotations — "After the UIKit migration is done, run the full test suite and fix any failures." For iOS teams doing large-scale refactors: queue 10-15 migration tasks Friday evening, review clean PRs Monday morning. Combine with Cursor's Rules for AI (.cursorrules) to ensure all background agents follow your team's coding conventions, import ordering, and documentation standards automatically.

Windsurf · Context Windsurf's "Memories" Feature: Persistent Project Context That Survives Session Restarts

Windsurf (formerly Codewind by Codeium) shipped a powerful feature that most developers haven't discovered: Memories — a persistent knowledge store that accumulates project-specific context across sessions. Unlike ephemeral chat history, Memories capture architectural decisions ("we use MVVM-C with Combine"), coding conventions ("all network calls go through NetworkService"), and past debugging insights ("the flaky test in AuthTests is caused by keychain race conditions"). The system auto-extracts these from your conversations and stores them in .windsurf/memories.md. The advanced usage: manually add "remember:" annotations in conversations to force-save critical context. For teams: commit .windsurf/memories.md to your repo so every team member's Windsurf agent inherits the same project knowledge — effectively creating a shared team brain that prevents repetitive explanations. This pattern applies identically to Claude Code's CLAUDE.md and Cursor's .cursorrules — the key insight is that persistent context files are the highest-ROI investment in AI-assisted development.
