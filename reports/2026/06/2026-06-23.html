Create ML · Fine-Tuning Fine-Tune On-Device Models With Create ML Components: The Federated Learning Shortcut

Create ML Components (introduced WWDC24, matured in Xcode 18) enables a powerful pattern most iOS developers overlook: on-device model fine-tuning using user-specific data that never leaves the device. The workflow: (1) Ship a base model via your app bundle. (2) Collect user-specific training data locally (e.g., the user's typing patterns, frequently accessed features, personal document layouts). (3) Use MLTrainingSession to fine-tune the model on-device during idle time (plugged in, charging, backgrounded). (4) Save the personalized model to the app's container. The result: each user gets a model adapted to their specific usage patterns — without sending training data to any server. For Binance: apply this to personalized anomaly detection — train a small classifier on each user's normal transaction patterns, then flag unusual activity locally before it reaches your backend fraud system. This provides sub-second alerting while maintaining total data privacy. The fine-tuning process takes 30-90 seconds for tabular models on iPhone 15+. Apple Docs →
